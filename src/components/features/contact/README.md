Contacto feature
=================

Este directorio contiene la implementación canonical del formulario de contacto.

- `ContactForm.tsx`: componente cliente con validación y envío a `/api/contact`.

Nota: anteriormente el componente vivía en `src/app/contacto/components`. Se ha movido
para seguir la convención de "features" y mantener `app/` solo para rutas/layouts.
